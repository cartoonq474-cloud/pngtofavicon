How to Add a Favicon to Your Website

Learn how to install your favicon packages on static hosting servers or custom CMS frameworks.

Step 1: Generate the Files

Upload your artwork to our homepage tool to generate the ZIP. Unzip the package to retrieve the files.

Step 2: Upload files to Server Root

Upload all extracted favicon files (favicon.ico, apple-touch-icon.png, etc.) directly into your website's main public root directory (usually public_html or public). E.g., they should be reachable at yoursite.com/favicon.ico.

Step 3: Paste Header Markup

Paste these standard links into the header area of all your page files:

<link rel="icon" type="image/x-icon" href="/favicon.ico"> <link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png"> <link rel="manifest" href="/site.webmanifest">
