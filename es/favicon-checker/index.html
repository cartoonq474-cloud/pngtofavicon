What's Included in Your Download

Every file you need for full cross-browser and cross-device favicon support

ICO

favicon.ico

The classic multi-size ICO format, containing 16×16, 32×32, and 48×48 icons. Required for legacy browser support including older versions of Internet Explorer.

16

favicon-16x16.png

Standard browser tab icon at 16×16 pixels. Used by most modern browsers as the primary tab favicon for standard-density displays.

32

favicon-32x32.png

High-DPI browser tab icon at 32×32 pixels. Displayed on Retina and HiDPI screens for crisp, sharp favicon rendering in browser tabs.

180

apple-touch-icon.png

Apple Touch Icon at 180×180 pixels for iPhone, iPad, and iPod Touch. Displayed when users add your website to their iOS home screen.

192

android-chrome-192x192.png

Android home screen icon at 192×192 pixels. Used when Android users add your site to their home screen via Chrome or other browsers.

512

android-chrome-512x512.png

High-resolution PWA icon at 512×512 pixels. Required for Progressive Web App install prompts and splash screens on Android devices.

JSON

site.webmanifest

Web App Manifest file containing icon references, theme color, and background color. Essential for PWA support and Android home screen integration.

Frequently Asked Questions

Why does the favicon checker say my icon is missing even if I see it?
Browsers often cache favicons heavily, so you might see an old cached icon even if the file is missing from your server, or if the HTML code has markup errors. Our checker requests the live HTML and parses it fresh to bypass local browser caches.

How can I fix a missing apple-touch-icon?
Generate a 180×180 PNG image, upload it to your server root as `apple-touch-icon.png`, and add `` inside your HTML header. You can use our homepage PNG converter to generate this file automatically.

Does a missing favicon affect search engine SEO?
Yes. Google Search displays website favicons next to search results in mobile and desktop viewports. If Google's bot cannot fetch your favicon, it displays a generic fallback icon, which can significantly lower your click-through rates (CTR).
